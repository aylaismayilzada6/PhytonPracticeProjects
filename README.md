# 100 Days of Python Projects

This repository contains a collection of projects developed as part of a self-initiated **"100 Days of Python"** challenge. The goal of this journey was to strengthen practical programming skills in Python through hands-on projects, covering topics such as APIs, automation, data processing, and object-oriented programming.

---

## 📌 Overview

Throughout this challenge, I built multiple mini-projects and applications to deepen my understanding of:

* Python fundamentals and advanced concepts
* Working with APIs (REST APIs, JSON handling)
* Automation scripts and real-world problem solving
* File handling (CSV, text files)
* Object-Oriented Programming (OOP)
* GUI development with Tkinter
* Game development using Python


## 🛠️ Technologies Used

* **Python 3**
* **Requests (API handling)**
* **Tkinter (GUI applications)**
* **CSV & File Handling**
* **Twilio API (messaging integration)**
* **News API / Stock API**
* Object-Oriented Programming (OOP)
